Claude Fable 5 System Prompt for Claude Code

Claude Fable 5 is a custom system prompt for Claude Code.

It is a single Markdown system prompt file designed to customize how Claude Code behaves in terminal-based coding sessions.

claude --dangerously-skip-permissions --system-prompt-file CLAUDE-FABLE-5-SYSTEM-PROMPT.md

Repository:

https://github.com/angelospillos/claude-fable-5-system-prompt

⸻

What Is This?

This repository contains a custom Claude Code system prompt:

CLAUDE-FABLE-5-SYSTEM-PROMPT.md

Use this file with the Claude CLI to load Claude Fable 5 as the active system prompt for Claude Code.

⸻

Quick Start

Clone the repository:

git clone https://github.com/angelospillos/claude-fable-5-system-prompt.git
cd claude-fable-5-system-prompt

Run Claude Code with the Claude Fable 5 system prompt:

claude --system-prompt-file CLAUDE-FABLE-5-SYSTEM-PROMPT.md

Run Claude Code with permission prompts skipped:

claude --dangerously-skip-permissions --system-prompt-file CLAUDE-FABLE-5-SYSTEM-PROMPT.md

⸻

Recommended Command

claude --dangerously-skip-permissions --system-prompt-file CLAUDE-FABLE-5-SYSTEM-PROMPT.md

Use this only in a trusted local development environment.

curl -O https://raw.githubusercontent.com/angelospillos/claude-fable-5-system-prompt/main/CLAUDE-FABLE-5-SYSTEM-PROMPT.md
